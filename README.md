# Supermarket Sales Data Analysis

# Overview

This project focuses on analyzing supermarket sales data using SQL and Python to identify sales trends, customer behavior, and product performance.

# Tools & Technologies

* SQL
* Python (Pandas, NumPy, Matplotlib)

# Project Workflow

1. Data Cleaning and Preparation
2. SQL-based Business Queries
3. Exploratory Data Analysis using Python
4. Data Visualization using Matplotlib

# Key Insights

* Identified top-performing product categories
* Analyzed monthly sales and profit trends
* Compared sales performance across regions
* Observed customer purchasing behavior patterns
